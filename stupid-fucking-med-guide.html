Chemistry

LABELLING CHEMICALS

A possible format you can use is as follows:

[CHEMICAL SHORTHAND] | [PURPOSE] | [EFFECT PER 5u] | [OVERDOSE AMOUNT / SIDE EFFECTS]

Side effects can mean that the chemical will cause patient to receive a certain damage type (Like how Phalanxamine heals 1 Genetic but also causes 1 Radiation damage).

For Example ( PYRAZINE ):

Pyra | Heat | 5u = 50 Heat | 20u OD

Calculating the effect per 5u is just multiplying the following:
if (0.5 units per second) = multiply healing value and 0.5 by 10
if (0.1 units per second) = multiply healing value and 0.1 by 50

You can see these values in the guidebook.

SUPERHEATING

Superheating is a concept where you set up a specific beaker to be able to instantly mix chemicals that need heat. This can be efficient when handling heat chemicals like Pyrazine and Lacerinol

In order to superheat a beaker, you need to apply 1u of either carbon or plasma into a beaker, then put it on a hotplate for 5-10 secs. This will set the beaker's temperature up to thousand kelvins.

The superheater beaker will:

  • Work when inserted in a chemmaster
  • Will NOT work when inserted in a chem dispenser
  • Work even if the 1u chemical was removed
  • Reset its temps when put in a chem dispenser and dispensed any chemical of any amount
RESET BEAKER TEMP

Putting the beaker on the chem dispenser and dispensing any chemical of any amount resets its temperature no matter what.
